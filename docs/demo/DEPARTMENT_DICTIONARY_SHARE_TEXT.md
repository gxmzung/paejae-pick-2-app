# Department Dictionary Share Text

## Short Version

배재Pick 2.0에 학과백과 기능을 추가하고 있습니다.

신입생, 전과 관심자, 타학과 학생이 학과의 기본 정보, 배우는 내용, 진로 방향, 건물 위치, 관련 나섬이 미션을 쉽게 볼 수 있도록 만드는 기능입니다.

현재는 기본 데이터와 화면 구조를 먼저 구현한 상태이고, 이후 학과별 인터뷰와 공식 확인을 받아 정확도를 높일 예정입니다.

## For Student Council / Senior Students

안녕하세요. 배재Pick 2.0 학과백과 기능을 만들고 있습니다.

학교 홈페이지식 소개가 아니라, 실제 학생들이 궁금해하는 학과 분위기, 1학년 때 중요한 것, 전과 관심자가 준비할 것, 진로 방향 등을 정리하려고 합니다.

현재는 기본 화면과 데이터 구조를 먼저 만든 상태이고, 추후 학과별로 짧은 인터뷰를 받아 반영하려고 합니다.

가능하다면 학과 소개 문구나 인터뷰 질문에 대해 의견 부탁드립니다.

## For Professors / Department Office

안녕하세요. 배재Pick 2.0 프로젝트에서 학과백과 기능을 개발하고 있습니다.

이 기능은 학생들이 학과 정보, 건물 위치, 전과 탐색 참고 정보, 학과별 안내를 더 쉽게 볼 수 있도록 돕는 캠퍼스 탐색 기능입니다.

현재 표시되는 학과 정보는 기본 mock data이며, 공식 정보와 인터뷰 기반 정보를 구분해서 표시하는 구조로 설계하고 있습니다.

추후 학과 공식 정보나 표현 기준에 대해 확인받고 싶습니다.
