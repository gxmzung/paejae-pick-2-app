# 배재Pick 2.0 v3.8 Department Dictionary Demo Package

## Goal

v3.8 prepares the Department Dictionary for sharing, explanation, and feedback collection.

This version does not add new app logic.

It packages the current Department Dictionary feature so it can be shown to:

- professors
- student councils
- department representatives
- senior students
- internal testers
- project teammates

## Current Feature State

The Department Dictionary currently includes:

- full basic department seed data
- department list screen
- search UI
- category filter chips
- department detail screen
- department card visual polish
- Nasumi / QR mission connection wording
- verification status labels

## 30-Second Explanation

배재Pick 학과백과는 학교 홈페이지를 복사한 기능이 아니라, 학생이 실제로 학과를 탐색할 수 있게 만든 캠퍼스 학과 가이드입니다.

신입생, 전과 관심자, 타학과 학생이 학과의 분위기, 배우는 내용, 진로 방향, 건물 위치, 관련 나섬이 미션을 한 번에 볼 수 있도록 설계했습니다.

현재 내용은 기본 seed data 단계라 확인 필요 상태이고, 이후 학과 학생회나 재학생 인터뷰를 받아서 신뢰도를 높일 계획입니다.

## One-Line Pitch

학과백과는 신입생·전과 관심자·타학과 학생이 학과를 더 쉽게 이해하도록 돕는 배재Pick의 캠퍼스 탐색 기능입니다.

## Feedback Questions

### For Professors

1. 학과 소개에 반드시 들어가야 할 공식 정보는 무엇인가요?
2. 학생들이 오해하지 않게 주의해야 할 표현이 있을까요?
3. 전과 관심자에게 안내해도 되는 조언 범위는 어디까지인가요?
4. 교수님 연구실/학과실 위치를 앱에 안내해도 괜찮을까요?
5. 학과 공식 자료와 학생 인터뷰를 어떻게 구분하면 좋을까요?

### For Student Councils

1. 학과 분위기를 한 문장으로 설명하면?
2. 신입생이 가장 많이 궁금해하는 것은 무엇인가요?
3. 학과 행사나 학생회 활동 중 소개할 만한 것이 있나요?
4. 학과실/주요 건물 안내를 넣어도 괜찮을까요?
5. 학과백과에 넣으면 안 되는 표현이 있을까요?

### For Senior Students

1. 1학년 때 가장 중요했던 과목이나 역량은?
2. 이 학과에 잘 맞는 학생은 어떤 유형인가요?
3. 전과/복수전공을 고민하는 학생에게 해주고 싶은 조언은?
4. 졸업 전 만들면 좋은 포트폴리오는?
5. 입학 전 알았으면 좋았을 현실적인 조언은?

## Safety / Trust Rule

Do not present unverified information as official.

Use labels:

- 확인 필요
- 재학생 인터뷰
- 학생회 확인
- 교수/학과실 확인
- 학교 공식 정보

## Current Limitation

The current version does not yet contain verified interviews.

It is a working prototype for collecting feedback and improving department information.
