# 배재Pick 2.0 v1.0 출시 범위

## 목표

v1.0은 완성형 공식 학교 앱이 아니라, Android 시장 반응을 보기 위한 실사용 가능 MVP이다.

목표는 재학생들이 실제로 설치하고 열어볼 이유가 있는 수준까지 만드는 것이다.

---

## 포함 기능

### 1. Splash / Login mock

- 앱 첫 화면
- 로그인 UI
- 이메일 인증 안내
- 실제 인증은 v1.1 이후

### 2. 홈

- 오늘의 Pick
- 오늘의 학생식당
- 오늘의 나섬이
- 오늘의 미션
- 동아리 모집 preview
- 도감 완성도

### 3. 학생식당 / CityBrain

- 오늘 메뉴
- 혼잡도 mock
- 예상 대기 시간
- 추천 방문 시간
- 만족도 버튼
- 오늘의 메뉴 카드 획득

### 4. 캠퍼스 도감

- 획득/미획득 카드 상태
- 카드 상세
- 도감 완성도 계산
- SharedPreferences 로컬 저장

### 5. 미션

- 코드 입력
- 카드 획득
- 미션 완료 화면

### 6. 학과 나섬이 투어

- 학과별 나섬이 카드
- 학과 건물 코드
- 학과 건물 안내
- 신입생/통학생/기숙사생 대상 캠퍼스 적응 기능

### 7. 동아리 공고관

- 동아리 목록
- 동아리 상세
- 모집 기간
- 활동 장소
- 지원 링크 mock
- 문의 링크 mock

### 8. 마이페이지

- 프로필 mock
- 도감 완성도
- 획득 카드
- 완료 미션
- 관심 동아리

---

## 제외 기능

- 실제 학교 이메일 인증
- 실제 QR 카메라 스캔
- Firebase/Supabase 서버 동기화
- 실제 위치 추적
- 실제 시설 신고
- 관리자 콘솔
- 결제/기프티카드 교환
- 학교 공식 데이터 연동
- 교수님 개인 연구실 상세 공개

---

## Android 출시 전략

1. 내부 테스트 APK/AAB 빌드
2. 지인/동아리/재학생 중심 테스트
3. 학생식당 기능 반응 확인
4. 도감/나섬이 기능 재방문성 확인
5. 동아리 공고관 수요 확인
6. 개인정보/학교명/마스코트 사용 범위 검토
7. 필요 시 학교 제안용 데모 버전 분리
