# 배재Pick 2.0 내부 테스트 체크리스트

## 설치 확인

- [ ] APK 설치 가능
- [ ] 앱 이름이 배재Pick 2.0으로 표시됨
- [ ] 앱 실행 가능
- [ ] Splash / Login 화면 표시

## 주요 기능 확인

- [ ] 학생식당 상세 화면 진입
- [ ] 메뉴 카드 획득 가능
- [ ] 도감 카드 목록 표시
- [ ] 코드 입력 가능
- [ ] 학과 나섬이 투어 화면 진입
- [ ] 동아리 상세 화면 진입
- [ ] 관심 동아리 저장 가능
- [ ] 개인정보 안내 화면 표시
- [ ] 로컬 데이터 초기화 가능

## 피드백 질문

- [ ] 학생식당 기능 때문에 앱을 열 이유가 있는가?
- [ ] 도감/나섬이 수집이 재방문을 유도하는가?
- [ ] 학과 나섬이 투어가 학교 구조 이해에 도움이 되는가?
- [ ] 동아리 공고관이 유용한가?
- [ ] 개인정보 안내가 부담스럽지 않은가?
