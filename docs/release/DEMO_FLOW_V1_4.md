# 배재Pick 2.0 v1.4 Demo Flow

## 90-Second Demo

### 0~10s: App Entry

배재Pick 2.0은 배재대 학생을 위한 스마트캠퍼스 MVP 앱입니다.

첫 설치 이유는 학생식당이고, 재방문 이유는 나섬이 도감과 미션입니다.

### 10~25s: Home

홈에서는 오늘의 학생식당, 미션, 학과 나섬이 투어, 동아리 공고를 한 번에 볼 수 있습니다.

### 25~40s: Cafeteria

학생식당 상세에서는 오늘 메뉴, 혼잡도 mock, 예상 대기 시간, 추천 방문 시간을 확인할 수 있습니다.

학생은 메뉴 카드를 획득할 수 있습니다.

### 40~55s: Mission

미션 화면에서는 QR 스캔 mock과 코드 입력 기능을 제공합니다.

내부 테스트에서는 빠른 테스트 코드를 눌러 미션 흐름을 검증할 수 있습니다.

### 55~70s: Department Nasumi Tour

학과 나섬이 투어는 학생이 자기 학과 건물 외의 캠퍼스 공간을 탐험하게 만드는 기능입니다.

신입생, 통학생, 기숙사생의 캠퍼스 적응을 돕습니다.

### 70~85s: Club Notice

동아리 공고관은 익명 게시판이 아니라 동아리 모집 정보를 구조화해 보여주는 공간입니다.

관심 동아리는 로컬에 저장됩니다.

### 85~90s: Privacy

현재 버전은 서버 없이 로컬 저장 중심으로 동작합니다.

실명, 학번, 전화번호, 실시간 위치를 수집하지 않습니다.
