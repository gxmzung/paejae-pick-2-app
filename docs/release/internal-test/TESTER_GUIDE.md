# 배재Pick 2.0 내부 테스트 안내문

## 1. 테스트 목적

배재Pick 2.0은 배재대학교 학생을 위한 스마트캠퍼스 MVP 앱입니다.

이번 내부 테스트의 목적은 앱의 완성도를 판단하는 것이 아니라, 실제 학생들이 이 앱을 설치하고 사용할 이유가 있는지 확인하는 것입니다.

## 2. 핵심 확인 포인트

- 학생식당 정보 때문에 앱을 열 이유가 있는가?
- 나섬이 도감과 미션이 다시 들어오게 만드는가?
- 학과 나섬이 투어가 캠퍼스 구조를 익히는 데 도움이 되는가?
- 동아리 공고관이 에브리타임이나 인스타보다 보기 쉬운가?
- 개인정보 안내가 부담스럽지 않은가?
- 앱 디자인이 학교 앱처럼 보이는가?

## 3. 현재 버전

배재Pick 2.0 v1.5 Internal Test Package

현재 버전은 내부 테스트용입니다.

공식 학교 앱이 아니며, 실제 학교 데이터와 연동되어 있지 않습니다.
