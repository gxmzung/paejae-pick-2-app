# 배재Pick 2.0 Android APK 설치 가이드

## 1. 설치 파일

내부 테스트용 APK 파일은 아래 경로에서 생성됩니다.

- Debug APK: build/app/outputs/flutter-apk/app-debug.apk
- Release APK: build/app/outputs/flutter-apk/app-release.apk

테스터에게는 보통 app-release.apk를 전달하는 것이 좋습니다.

## 2. APK 파일 전달 방법

APK 파일은 다음 방식으로 전달할 수 있습니다.

- 카카오톡 파일 전송
- 이메일 첨부
- Google Drive 공유
- USB 또는 직접 파일 복사

## 3. Android 기기에서 설치하는 방법

1. Android 기기에서 APK 파일을 다운로드합니다.
2. 다운로드한 APK 파일을 터치합니다.
3. "보안상의 이유로 이 출처의 알 수 없는 앱을 설치할 수 없습니다"라는 안내가 나오면 설정으로 이동합니다.
4. "이 출처 허용" 또는 "알 수 없는 앱 설치 허용"을 켭니다.
5. 다시 APK 파일을 열고 설치를 진행합니다.
6. 설치가 끝나면 앱 목록에서 "배재Pick 2.0"을 실행합니다.

## 4. 설치가 안 될 때 확인할 것

- 기존에 설치된 배재Pick 2.0 앱이 있으면 삭제 후 다시 설치합니다.
- 저장 공간이 부족하지 않은지 확인합니다.
- APK 파일이 완전히 다운로드되었는지 확인합니다.
- Android 버전이 너무 낮지 않은지 확인합니다.
- 회사/학교 보안 정책이 설치를 막고 있지 않은지 확인합니다.

## 5. 테스트 후 삭제 방법

1. Android 앱 목록에서 배재Pick 2.0을 길게 누릅니다.
2. 앱 정보 또는 삭제를 선택합니다.
3. 삭제를 누릅니다.

앱을 삭제하면 기기 내부에 저장된 도감, 관심 동아리, 미션 상태도 함께 삭제될 수 있습니다.

## 6. 주의 사항

현재 앱은 공식 학교 앱이 아닙니다.

앱 안의 메뉴, 혼잡도, 동아리 정보는 테스트용 mock 데이터입니다.

실제 학교 이메일 인증, 실제 QR 스캔, 실제 위치 추적, 실제 민원 신고 기능은 아직 포함되어 있지 않습니다.
