# 배재Pick 2.0 개인정보처리방침 초안

## 1. 목적

배재Pick 2.0은 배재대학교 학생을 위한 스마트캠퍼스 MVP 앱입니다.

본 문서는 내부 테스트 및 초기 배포 준비를 위한 개인정보 처리 기준 초안입니다.

## 2. 수집하는 개인정보

v1.0/v1.2 기준으로 배재Pick 2.0은 서버에 개인정보를 저장하지 않습니다.

현재 앱은 다음 정보를 기기 내부에만 저장합니다.

- 도감 카드 획득 상태
- 미션 완료 상태
- 관심 동아리 저장 상태
- 메뉴 카드 획득 상태

## 3. 수집하지 않는 정보

현재 버전에서는 다음 정보를 수집하지 않습니다.

- 실명
- 학번
- 전화번호
- 주민등록번호
- 실시간 위치 정보
- 식당 방문 이력
- 시설 신고자 신원
- 교수님 연구실 방문 기록

## 4. 저장 위치

현재 저장 정보는 사용자의 기기 내부에 저장됩니다.

서버 전송은 이루어지지 않습니다.

## 5. 데이터 삭제

사용자는 앱의 마이페이지에서 로컬 데이터를 초기화할 수 있습니다.

## 6. 향후 변경 가능성

향후 학교 이메일 인증, 서버 동기화, QR 스캔, 관리자 기능이 추가될 경우 개인정보처리방침은 별도로 개정됩니다.
