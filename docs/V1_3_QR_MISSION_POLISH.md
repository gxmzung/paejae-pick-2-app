# 배재Pick 2.0 v1.3 QR Mission Polish

## Goal

v1.3 improves the mission experience.

This version does not add a real camera QR scanner yet.  
Instead, it adds a QR scan mock screen and improves the code-based mission flow for internal testing.

## Why

The mission system is one of the main repeat-use features of Paejae Pick.

Students should quickly understand:

- what code to enter
- what reward they can get
- whether QR scanning is real or mock
- which missions are currently available

## Scope

- QR scan mock screen
- Mission guide section
- Quick test code buttons
- Better mission success/failure feedback
- Mission code list for internal testing

## Not Included

- Real camera permission
- Real QR scanner
- Backend mission validation
- Anti-abuse logic
- GPS verification
