# GitHub Release Notes - v1.0 Paejae Pick Release Candidate

## Title

v1.0 Paejae Pick 2.0 Release Candidate

## Summary

This release is the first Android-focused MVP release candidate for Paejae Pick 2.0.

Paejae Pick 2.0 is a Flutter-based smart campus app concept for Paichai University students.

The first reason to install the app is cafeteria information.  
The reason to return is Nasumi collection, campus missions, department tours, and club notices.

## Included

- Splash / Login mock
- Home
- Cafeteria detail
- Congestion mock
- Menu card acquisition
- Code-based mission
- Collection / card detail
- Department Nasumi Tour
- Club notice detail
- Favorite club local storage
- Privacy guide
- Local data reset
- Release candidate documentation

## Not Included Yet

- Real school email authentication
- Real QR camera scanning
- Firebase/Supabase sync
- Real-time location tracking
- Admin console
- Official school data integration
- Release signing configuration
