# Paejae Pick 2.0 v0.8 Club Notice Detail

## Goal

v0.8 improves the club notice feature.

The club notice module is not an anonymous board.  
It is a structured official-style recruitment notice space for clubs and project teams.

## Problem

Club recruitment information is often scattered across Everytime, Instagram, KakaoTalk, posters, and personal networks.

Students, especially freshmen, may miss recruitment opportunities because the information is not structured or persistent.

## Solution

Paejae Pick provides a club notice hall.

Each club has a structured recruitment notice with:

- club name
- category
- recruiting status
- deadline
- target members
- activity place
- activity day
- beginner availability
- required skills
- apply link mock
- contact link mock

## v0.8 Scope

- Club notice detail screen
- Clickable club cards
- Favorite club local storage
- My Page favorite club count integration
- No comments
- No anonymous board
- No open community posting

## Privacy

v0.8 stores favorite clubs locally only.

No server sync is used.
