# 배재Pick 2.0 v1.0 Release Candidate

## 1. 목표

v1.0은 Android 시장 반응을 보기 위한 배재Pick 2.0의 첫 출시 후보 버전이다.

완성형 공식 학교 앱이 아니라, 실제 재학생들이 설치해서 학생식당, 도감, 미션, 학과 나섬이, 동아리 공고관 기능을 체험해볼 수 있는 MVP를 목표로 한다.

---

## 2. 핵심 전략

배재Pick의 첫 설치 이유는 학생식당이다.

배재Pick의 재방문 이유는 나섬이 도감, 미션, 학과 나섬이 투어, 동아리 공고관이다.

---

## 4. v1.0 제외 기능

v1.0에서는 아래 기능을 의도적으로 제외한다.

- 실제 학교 이메일 인증
- Firebase/Supabase 서버 동기화
- 실제 QR 카메라 스캔
- 실시간 위치 추적
- 실제 학생 제보
- 운영 예시 화면
- 결제/기프티카드 교환
- 학교 공식 데이터 연동
- POS/키오스크 연동
- YOLO 학생 체감 혼잡도 연동
- 교수님 개인 연구실 상세 공개

---

## 5. 개인정보 원칙

v1.0은 서버에 개인 데이터를 저장하지 않는다.

로컬 저장 대상:

- 획득 카드
- 관심 동아리
- 미션 완료 상태

저장하지 않는 것:

- 실명
- 정확한 학번
- 전화번호
- 실시간 위치
- 식당 방문 이력
- 민원 신고자 신원

---

## 6. Android 테스트 기준

v1.0은 Android 내부 테스트를 우선한다.

테스트 대상:

- 재학생 5~10명
- 동아리 운영진
- 기숙사생
- 통학생
- 신입생 또는 저학년

확인할 질문:

- 학생식당 기능 때문에 앱을 열 이유가 있는가?
- 도감/나섬이 기능이 재방문을 유도하는가?
- 학과 나섬이 투어가 캠퍼스 구조 학습에 도움이 되는가?
- 동아리 공고관이 에브리타임보다 정리되어 보이는가?
- 개인정보 안내가 부담스럽지 않은가?
