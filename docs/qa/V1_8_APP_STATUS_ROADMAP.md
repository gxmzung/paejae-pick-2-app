# 배재Pick 2.0 v1.8 App Status / Roadmap

## Goal

v1.8 adds an in-app status and roadmap screen.

The goal is to clearly explain what is implemented, what is still mock, and what must be done before a real release.

## Scope

- App status screen
- Implemented feature list
- Not-yet-implemented feature list
- Release roadmap
- My Page entry point

## Why

Internal testers may misunderstand mock features as broken features.

The app should explain its current MVP state directly inside the UI.

## Current Status

Paejae Pick 2.0 is currently an internal-testable Flutter MVP.

It is not an official university app yet.
