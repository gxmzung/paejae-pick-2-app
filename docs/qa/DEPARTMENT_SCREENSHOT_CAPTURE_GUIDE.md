## Capture Rules

- Use the same device for all screenshots
- Use portrait mode
- Use light mode
- Hide personal notifications
- Keep status bar visible
- Do not crop UI elements
- Capture after the screen fully loads

## Review Questions

1. Does this look close to the generated design direction?
2. Is the screen understandable within 3 seconds?
3. Is there too much text?
4. Is the button action obvious?
5. Would this screenshot be convincing in an Instagram story or DM?
6. Is any unverified information presented too confidently?
