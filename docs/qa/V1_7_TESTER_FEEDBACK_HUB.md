# 배재Pick 2.0 v1.7 Tester Feedback Hub

## Goal

v1.7 adds an in-app tester feedback hub.

The goal is to make internal testing easier by showing the key feedback questions inside the app.

## Scope

- Tester feedback screen
- Test criteria explanation
- Install reason questions
- Repeat-use questions
- Campus exploration questions
- Club notice questions
- Privacy/trust questions
- My Page entry point

## Core Question

```text
이 앱을 실제로 설치해서 계속 쓸 이유가 있는가?
Why

External feedback forms are useful, but internal testers often do not know what to focus on.

The app now includes a built-in guide so testers can evaluate the MVP from the intended product perspective.

Not Included
Remote feedback submission
Google Forms integration
Server-side feedback storage
Analytics tracking
