# 배재Pick 2.0 v3.3 Department Full Seed Basic

## Goal

v3.3 expands the Department Dictionary from selected seed departments to a full basic department list.

## Scope

This version adds basic department entries only.

It does not add verified interviews yet.

## Added Structure

Each department includes:

- department name
- college or group
- building placeholder
- department code
- keywords
- one-line intro
- learn topics
- career direction
- transfer note
- related Nasumi name
- QR mission code
- verification status

## Verification Status

All department data is marked as 확인 필요.

## Why Interview Is Later

Interview content is more sensitive than basic seed data.

It should be added only after:

- student council confirmation
- department representative confirmation
- professor or department office confirmation when needed
- permission for public display

## Product Direction

The Department Dictionary should first cover all departments at a basic level.

After that, high-demand departments can be upgraded with interviews and verified details.

## Next Step

v3.4 should improve Department Dictionary filtering, grouping, and UI readability.
