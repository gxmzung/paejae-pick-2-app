# 배재Pick 2.0 v3.6 Department Search / Filter UI

## Goal

v3.6 connects the Department Dictionary list to actual search and filter UI.

## Scope

- Converts Department Intro screen into a stateful searchable screen
- Adds search field
- Adds category filter chips
- Adds result count
- Adds empty result state
- Keeps filter guide screen as an explanation page

## Search Targets

Search checks:

- department name
- college/group
- building
- department code
- keywords
- career paths

## Filter Groups

- 전체
- 인문사회
- 자연과학
- 공학
- 예체능
- 평생교육

## Product Value

Students should not scroll through a long department list manually.

The Department Dictionary should help students quickly answer:

- What department am I looking for?
- Which department fits my interest?
- Which department is related to my target career?
- Where is the department located?

## Safety Rule

This version does not add interviews.

All unverified department information remains 확인 필요.
