# 배재Pick 2.0 v4.0 Department UI Fix Pass

## Goal

v4.0 is the first UI correction pass after the Department Dictionary QA package.

This version does not add interviews or new large features.

It focuses on making the Department Dictionary easier to show, capture, and explain.

## Fix Direction

### 1. Department Dictionary Main

- keep search and filter visible near the top
- reduce unnecessary text density
- make first screen screenshot-friendly
- keep Nasumi placement consistent

### 2. Department Cards

- department name should be the strongest text
- category / building / mission code should be readable
- status labels should not dominate the card
- CTA should be clear

### 3. Department Detail

- recommendation section should be easy to scan
- learning topics and career paths should feel like cards or bullets
- admission/transfer notes should be clearly marked as reference information
- Nasumi and QR mission should look like a connected campus activity

### 4. Trust / Safety

All unverified department data remains marked as 확인 필요.

Official information and interview-based information must stay separated.

## Definition of Done

- debug APK builds successfully
- Department Dictionary main is screenshot-ready
- Department Detail is understandable within 3 seconds
- no unverified content is presented as official
- README is updated
