# 배재Pick 2.0 v1.9 Club Notice Submit Mock

## Goal

v1.9 adds a mock flow for club notice submission.

The goal is to make the club notice hall feel like a structured recruitment system, not an anonymous board.

## Why

Paichai University currently does not have a strong centralized club recruitment platform.

Instead of copying anonymous boards, Paejae Pick should provide a clean notice-based structure.

## Scope

- Club notice submit request mock
- Club name field
- Recruitment title field
- Recruitment category field
- Contact field
- Description field
- Submit confirmation
- No backend storage yet

## Product Principle

This is not a free anonymous board.

It is a controlled notice hall for clubs.

## Not Included

- Real admin approval
- Server-side submission
- File upload
- Image upload
- Push notification
- Abuse reporting

## Future

- Admin console approval
- Club manager account
- School email verification
- Notice status: pending / approved / rejected
