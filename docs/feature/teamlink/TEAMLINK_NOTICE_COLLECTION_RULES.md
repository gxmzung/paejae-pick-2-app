# TeamLink Notice Collection Rules

## Purpose

This document defines how TeamLink should treat contest and program notices.

## Source Priority

### Priority 1: Official University Notice

Examples:

- 배재대학교 일반공지
- 학과 공지
- 대학일자리플러스본부 공지
- 창업지원 프로그램 안내

### Priority 2: Official External Contest Page

Examples:

- contest organizer page
- government or public institution contest page
- company-hosted official contest page

### Priority 3: Student Shared Notice

Examples:

- student finds contest and submits it
- team leader creates a recruitment post
- club shares contest information

## Display Rule

Each notice should display its trust level.

Recommended badges:

- 학교 공식
- 외부 공식
- 학생 제보
- 확인 필요

## Required Fields

- title
- source
- source_url
- deadline
- field
- organizer
- team_allowed
- required_roles
- status
- verification_status

## Status

- 모집 중
- 마감 임박
- 마감
- 확인 필요

## Important Limitation

TeamLink does not replace the official university website.

It helps students discover notices and form teams faster.

For final application details, students must check the official source.
