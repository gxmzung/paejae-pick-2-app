# 배재Pick 2.0 v4.1 TeamLink Concept

## Feature Name

배재 팀링크

## Proposal Title

배재 팀링크: 공모전 팀원 매칭을 통한 학생 소통·협력 활성화 방안

## Problem

Many students want to join contests, projects, or idea competitions but give up because they cannot find teammates.

Common cases:

- can develop but cannot design
- can present but struggles with planning
- can research but does not want to lead
- wants to join a contest but has no cross-major network
- does not know which contests are currently open
- misses announcements because notices are scattered

Current channels are fragmented:

- Everytime
- KakaoTalk rooms
- personal connections
- department announcements
- late or inconsistent notice sharing

## Core Goal

TeamLink is not just a team recruitment board.

The goal is to connect students by:

- contest interest
- role
- skill
- major
- available time
- participation status

## Key Value

- increase contest participation
- activate cross-major collaboration
- strengthen student capability
- improve university external performance
- give students a reason to open the app

## Why App Instead of Website

An app is more suitable because:

- students can check contest notices quickly
- deadline alerts can create a reason to return
- role-based matching can be linked to profile data
- students can browse team openings during spare time
- mobile access is easier than a separate website

## MVP Scope

### 1. Contest Notice Hub

- contest title
- field
- deadline
- organizer
- link
- team size
- required roles

### 2. Team Recruitment Card

- team name
- contest target
- current members
- needed roles
- short description
- contact method
- status

### 3. Role Tags

- 기획
- 개발
- 디자인
- 발표
- 자료조사
- 영상
- 문서
- 팀장
- AI
- 하드웨어
- 마케팅

### 4. Student Interest Profile

- interested fields
- possible roles
- available time
- preferred collaboration style
- portfolio link optional

## Important Limitation

The app cannot guarantee team quality.

It cannot handle all disputes, responsibility, or official contest submission.

The university or operator still needs rules for:

- inappropriate posts
- fake recruitment
- conflict handling
- privacy
- official contest notice verification

## Product Principle

TeamLink should lower the first barrier.

It should help a student move from:

"I want to join a contest but I have no team"

to:

"I found a team or posted my role."
