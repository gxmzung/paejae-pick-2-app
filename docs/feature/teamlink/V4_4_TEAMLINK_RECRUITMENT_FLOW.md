# 배재Pick 2.0 v4.4 TeamLink Recruitment Flow

## Goal

v4.4 adds the first recruitment detail and create recruitment mock flow for TeamLink.

## Scope

This version adds:

- team recruitment detail screen
- create team recruitment mock screen
- create recruitment entry button
- recruitment card navigation to detail screen

## User Flow

### 1. Browse TeamLink

Student opens TeamLink and sees:

- official notice source
- role tags
- contest notices
- team recruitment cards

### 2. Open Recruitment Detail

Student taps a team recruitment card.

The detail screen shows:

- team name
- target contest
- current members
- needed roles
- short team description
- status
- safety notes

### 3. Create Recruitment Mock

Student taps “팀 모집글 만들기”.

The mock screen shows:

- contest selection
- team name
- current member count
- contact method
- needed roles
- writing guide

## Important Limitation

This is still a mock flow.

It does not yet include:

- real account system
- real submit button
- backend save
- moderation
- report system
- notification
- official notice sync

## Safety Rule

Students must check the official contest notice before applying.

TeamLink only lowers the first barrier to forming a team.

It does not guarantee team quality or contest acceptance.
