# 배재Pick 2.0 v4.2 TeamLink Mock Screen

## Goal

v4.2 implements the first TeamLink mock screen inside the app.

## Scope

This version adds:

- TeamLink screen
- contest notice mock cards
- team recruitment mock cards
- role tag chips
- basic stats
- Participation Hub entry point

## Product Purpose

TeamLink helps students who want to join contests but cannot find teammates.

It connects students by:

- contest interest
- role
- department
- participation status
- deadline

## Main Sections

### 1. Hero

Explains TeamLink as a way to find contest teammates.

### 2. Role Tags

Shows possible roles:

- 기획
- 개발
- 디자인
- 발표
- 자료조사
- 문서
- AI
- 데이터
- 영상
- 팀장
- 하드웨어
- 마케팅

### 3. Contest Notices

Shows mock contests with:

- title
- field
- deadline
- organizer
- required roles
- status

### 4. Team Recruitment

Shows mock teams with:

- team name
- target contest
- current members
- needed roles
- short description

## Important Limitation

This is a mock screen.

It does not yet include:

- real backend
- real user profile
- application workflow
- moderation system
- official contest verification
- notification system

## Safety Rule

TeamLink should never promise team quality.

It only lowers the first barrier to finding teammates.
