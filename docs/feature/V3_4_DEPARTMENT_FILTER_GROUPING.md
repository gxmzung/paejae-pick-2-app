# 배재Pick 2.0 v3.4 Department Filter / Grouped Browsing

## Goal

v3.4 defines the browsing structure for the Department Dictionary after full basic department seed data has been added.

The goal is to make many departments easy to search, filter, and compare.

## Why

After v3.3, the Department Dictionary includes many departments.

A simple long list is not enough.

Students need to quickly find departments by:

- 계열
- 관심 분야
- 건물
- 전과 관심도
- 진로 키워드
- 인터뷰 상태

## Primary Filters

### 전체

Shows all departments.

### 인문사회

Examples:

- 경영학과
- 행정학과
- 심리상담학과
- 항공서비스학과
- 미디어콘텐츠학과

### 자연과학

Examples:

- 간호학과
- 식품영양학과
- 생명공학과
- 보건의료복지학과

## Search Targets

Search should match:

- department name
- college/group
- keywords
- career paths
- building
- department code

## Department Card Display

Each department card should show:

- 학과명
- 계열 / 학부
- 대표 키워드
- 주요 건물
- 정보 상태
- 관련 나섬이
- 전과 관심자 참고 여부

## State Badges

Use badges:

- 확인 필요
- 인터뷰 예정
- 인터뷰 진행 중
- 인터뷰 완료
- 학교 공식 정보
- 학생회 확인

## Product Rule

Do not overload the Department Dictionary with long text on the list screen.

The list screen should answer:

- 어떤 학과가 있나?
- 어느 계열인가?
- 무엇을 배우나?
- 어디에 있나?
- 자세히 볼 가치가 있나?

Long explanations should stay in the detail screen.
