# 배재Pick 2.0 v2.6 Club Directory Seed Data

## Goal

v2.6 updates the Campus Participation Hub with a larger club directory seed dataset.

## Source

The seed data is based on a captured campus club list.

Only club names are used at this stage.

## Important Limitation

The app does not claim that the listed recruitment details are official.

Most fields are mock placeholders and require club/operator verification.

## Added Direction

- More realistic club list
- Sports clubs
- Culture clubs
- Volunteer/support clubs
- Study/academic clubs
- Mock recruitment details

## Excluded

Personal-looking names were not added until verified as actual club names.

## Next Step

Collect verified club data:

- Official club name
- Category
- Representative contact
- Activity place
- Activity day
- Recruitment period
- Beginner availability
- Description
