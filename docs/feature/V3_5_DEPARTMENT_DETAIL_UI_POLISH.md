# 배재Pick 2.0 v3.5 Department Detail UI Polish

## Goal

v3.5 improves the Department Detail screen so it feels closer to the generated Department Dictionary design concept.

## Scope

- Polishes hero wording
- Adds recommendation section
- Improves transfer/admission note wording
- Clarifies Nasumi and QR mission connection
- Makes the detail screen more useful for freshmen, transfer-interested students, and campus explorers

## Detail Screen Structure

The Department Detail screen should communicate:

1. What this department is
2. Who it fits
3. What students learn
4. Where students can go after graduation
5. What to prepare before entering or transferring
6. Which building/Nasumi/QR mission is connected

## UX Principle

The Department Dictionary should not feel like a static school website.

It should feel like a student-friendly campus guide.

## Safety Rule

All unverified department content remains marked as 확인 필요.

Interview content is not added in this version.
